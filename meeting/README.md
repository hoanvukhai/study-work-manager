# Meeting — Biên bản họp với GVHD

Thư mục này lưu **biên bản tất cả các buổi họp** với Giảng viên hướng dẫn (GVHD).  
Ghi lại ngay sau mỗi buổi họp — đừng để lâu sẽ quên.

---

## Quy tắc đặt tên file

```
YYYY-Wxx_[chu-de].md

Ví dụ:
  2026-W30_kickoff.md          — Buổi họp đầu tiên (kickoff), tuần 30 năm 2026
  2026-W32_review-research.md  — Họp review tài liệu Research
  2026-W35_review-db-design.md — Họp review thiết kế DB
```

> Dùng tuần (Wxx) thay vì ngày để dễ sắp xếp theo thứ tự.  
> Xem tuần hiện tại: `date +%V` (Linux/Mac) hoặc tra trực tuyến.
